# Student Performance Analysis

## 📌 Overview
This project generates a synthetic dataset of student marks and analyzes performance trends across subjects and genders using Python. It demonstrates data generation, statistical analysis, and visualization techniques.

## ⚡ Features
- Synthetic dataset of 1000 students
- Subject-wise performance histograms
- Gender-based comparisons
- Total marks distribution

## 🛠️ Requirements
- Python 3.8+
- NumPy
- Pandas
- Matplotlib
